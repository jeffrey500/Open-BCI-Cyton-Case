# Open-BCI-Cyton-Case
3D-Printed Case for the Open BCI Cyton Board, specifically for use with gel electrodes and a 4 AA battery pack.

<img width="50%" alt="case (1)" src="https://github.com/user-attachments/assets/8bdf8e3a-0da6-4b85-9fd6-4194b4122f5f" />

### Mounting
2 Velcro straps will be used to secure the base_case to a larger Velcro strap that goes across the subject's chest.

<img width="50%" alt="on_subject (1)" src="https://github.com/user-attachments/assets/e03e6e90-469e-4d2a-8ca5-38119946f39d" />

### Design for Printability
The case is designed to be printed in 4 separate components:
1. top_lid
2. top_case
3. base_case
4. base_lid

This allows for the case to be printed such that there are minimal overhangs, leading to a cleaner and stronger print.
It is recommended that the base_case and top_case be super-glued together.
