# Open-BCI-Cyton-Case
3D-Printed Case for the OpenBCI Cyton Board, specifically for use with gel electrodes and a 4 AA battery pack.  
This case ensures that the Cyton Board and battery are secure from tampering + eletrically isolated while still providing full access to the IO pins and interfaces of the Cyton Board.

## Mounting
2 Velcro straps will be used to secure the base_case to a 
larger Velcro strap that goes across the subject's chest.

## Design for Printability
The case is designed to be printed in 4 separate components:
1. top_lid
2. top_case
3. base_case
4. base_lid

This allows for the case to be printed such that there are minimal overhangs, leading to a cleaner and stronger print.
It is recommended that the base_case and top_case be super-glued together.

This case was printed on the Bambu Labs A1 Mini without supports without issue.

## Case Latches
There are small protrusions on the top_case such that the housing securely mates with the indents found on the top_lid. 
Similarly, there is a protrusion on the base_lid such that it also mates with the base_case, keeping the battery secure.
